# MATSU-bed blinky

Sample program of MATSU-bed

https://os.mbed.com/teams/MATSU-bed/wiki/MATSU-bed-config

tested on macOS Sierra (10.12.5)

## Usage

* Build

```
make
```

* Install

```
make install
```

## Requirements

* Python Package
  * intelhex
